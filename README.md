## Запуск проекта

Проект можно запустить с помощью команд:

```bash
composer install
```

```bash
php artisan serve
```

## Архитектура
Выбран фреймворк Laravel. Он обеспечивает быструю разработку, читаемый и поддерживаемый код, а также удобную структуру для масштабирования.

## Что было реализовано
- Разделение на контроллеры, ресурсы и интеграции
- Маршрутизация по API
- Реализовал интеграцию с LeedBook
- Ответы вывел все в виде ресурсов
- Написал SWAGGER документацию (использовал L5-Swagger)
